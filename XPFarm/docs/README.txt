Manual
====================

English
-----------
1. Start the application.
2. Set a maximum duration for the program to run, or, if you prefer, configure a key to stop the process.
3. Launch the program and switch to the Fortnite window.
4. The application will begin simulating movements automatically.


Español
-----------
1. Inicia la aplicación.
2. Configura una duración máxima en la que ejecute el programa o, si lo prefieres, configura si quieres usar una tecla para detener el proceso.
3. Inicia el programa y cambia a la ventana de Fortnite.
4. La aplicación comenzará a simular movimientos automáticamente.
